CSC321 Concurrent Programming Practical

Semaphores

The aim of this practical is to gain experience with using semaphores. In this practical you should make use of the Semaphore class in the Java Concurrent Library (import java.util.concurrent.*;). Note that in this class the method corresponding to up() is called release() and the method corresponding to down() is called acquire(). It is important that you study and experiment with each program.
  1. Implement in Java a semaphore solution to the Producer-Consumer problem. You should assume one producer, one consumer and a circular buffer.
